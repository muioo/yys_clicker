# 鼠标自动点击脚本

自动按时间间隔点击屏幕的Python脚本。

## 功能特性

- 每秒点击屏幕中心1次
- F6 快捷键启动/停止
- ESC 快捷键退出程序
- 控制台实时输出点击日志

## 环境要求

- Python 3.7+
- conda 虚拟环境 `yysscript`

## 安装

### 1. 激活虚拟环境

```bash
conda activate yysscript
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```

### 快捷键

| 按键 | 功能 |
|------|------|
| F6   | 启动/停止自动点击 |
| ESC  | 退出程序 |
